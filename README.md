# Your-Move  
A simple and efficient aim-training web application designed to improve reaction time and accuracy. The game presents targets that appear randomly on the screen, and the player must click them as quickly as possible.

## Live Demo  
https://aim-trainer-game-7xwo-dualite.netlify.app/

## Overview  
Your-Move is a browser-based aim trainer built using modern frontend tooling. The project focuses on clean UI, fast performance, and smooth interactions. This repository contains the full source code along with configuration files for development and deployment.

## Features  
- Targets spawn randomly for practicing accuracy and reaction time  
- Score tracking based on successful hits  
- Simple and minimal UI  
- Responsive layout suitable for desktop and mobile  
- Fast development workflow with Vite  

## Tech Stack  
- TypeScript  
- Vite  
- HTML  
- CSS (Tailwind CSS)  
- Netlify (deployment)

## How to Play  
1. Open the application in your browser.  
2. Start the game.  
3. Click the targets as they appear.  
4. Your score increases based on reaction accuracy.
